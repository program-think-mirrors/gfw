“自由门”是老牌的翻墙工具，由法轮功背景的美国公司研发。

官方网站是 动态网
http://www.dongtaiwang.com/
（必须翻墙才能访问）

软件本身是“绿色软件”，无需安装，无需管理员权限。
注：
7.6.9 之后的版本，启动时会尝试安装某个【网络驱动】，以提升翻墙效率。
如果你以【非】管理的身份运行该软件，启动时会要求你输入管理员密码（“安装驱动”需要管理员权限）。
如果你直接忽略（【不】输管理员密码），依然可以正常使用。

压缩包内仅有单个可执行文件，带有【数字签名】。
为了保险起见，解压缩之后，验证一下数字签名。

更详细的介绍，请参见俺的博文《“如何翻墙”系列：自由门——TOR 被封之后的另一个选择》
